# UI Visual Proposals (Modern SaaS Bento)

Updated: 2026-03-04

## Proposal A (Default): Calm Teal Bento
- Visual goal: Trust-first public data experience.
- Palette: `#08727f`, `#0d8a98`, `#27c2a8`, `#e8f4f5`, `#0f2d36`.
- Typography: Avenir Next + IBM Plex Sans fallback.
- Layout tone: Soft gradient shell + high legibility KPI cards + clear tab segmentation.
- Best fit: Public transparency dashboards with map + tabular detail views.

## Proposal B: Slate + Neon Accent Bento
- Visual goal: Higher technical energy while keeping data readability.
- Palette: Deep slate base + cyan accent rails.
- Typography: Same hierarchy, stronger contrast and tighter spacing.
- Layout tone: Data-heavy card framing + stronger active states.
- Best fit: Advanced analytics audiences.

## Proposal C: Warm Editorial Bento
- Visual goal: Narrative-first and policy communication oriented.
- Palette: Warm neutrals + restrained teal action color.
- Typography: Strong heading rhythm and section storytelling.
- Layout tone: Editorial blocks + softer table framing.
- Best fit: Stakeholder reporting and policy explainers.

## Final Decision
- **A / Calm Teal Bento** was confirmed as the final visual direction on 2026-03-04.
- Implementation baseline is locked to A in both:
  - HTML export dashboard
  - Streamlit dashboard
